# FOODIEPIE-BACKEND

This project contain a backend content for my project called foodiepie

## This Project Build with
* Node.js

## Installed dependencies
* Express Js
* MySQL
* Body Parser
* Cors
* Env
* Dotenv
* Morgan
* Nodemon
* qs
	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
